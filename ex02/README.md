### 追加機能
- 全消去：[del]を押すと全ての文字が消えるようにした。
- 1文字消去：[🔙]を押すと後ろの一文字が消えるようになった
- 四則演算：[+-*/]を押すと計算できるようにした、足し算、引き算、掛け算、割り算ができる
- べき乗：[**]を押すとべき乗できるようにした、例 3**3=
